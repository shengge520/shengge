# 第六题：flex还有哪些属性?

  1. flex: 1; 主要用于自适应布局，内容区会自动放大占满剩余空间
包含：
a, flex-grow: 1;
b, flex-shrink: 1;
c, flex-basis: 0%;

  1. flex-direction 默认row,属性值有column

  2. flex-wrap 是否换行

  3. flex-flow是flex-direction与flex-wrap的简写

  4. flex-grow 放大比例，默认是0

  5. flex-shrink 缩小比例，默认是1，负值对该属性无效

  6. flex-basis 该属性决定了在分配多余空间之前，item占据的主轴空间
