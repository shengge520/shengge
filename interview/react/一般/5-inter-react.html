# state与props区别？

  1. state可以看作组件的私有属性，用于组件内部的数据传递
  2. props是组件对外的接口，用于组件之间的传值，props在组件内部是不可修改，可以看作只读属性

扩展

  1. setState是异步更新，同步执行。因为react中会将多个setState合并在一起处理，从而提升程序的性能，这就是react高明的其中一点
  2. 异步，就有回调函数 setState()函数有两个参数，第一个对象赋值，第二个就是它的回调啦。

this.setState({count:this.state.count+1}, ()=>{ console.log("count ",this.state.count); });
