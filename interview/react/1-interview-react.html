# react生命周期

主要分为挂载、渲染、卸载

    1. 挂载 constructor() 初始化数据 componentWillMount() 服务端渲染(少) render componentDidMount()

  • 2.渲染 更新数据 shouldComponentUpdate(nextProps,nextState) 相当于[pureComponent] 组件是否需要更新 默认为true componentWillUpdate (nextProps,nextState) 组件将要更新 render componentDidUpdate(prevProps,prevState) 组件更新完成

# 1.1 shouldComponentUpdate 为false 只执行 shouldComponentUpdate 渲染时候其他不执行

场景，父点击btn给子传参，子需要更新就设置为true(默认),不需要则设置为false
